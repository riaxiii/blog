สวัสดีค่ะ\
\
\
อิเคดะ เทเรสะ ค่ะ\
\
จากที่มีประกาศในโนกิซากะโคจิจู\
ซิงเกิ้ลที่ 41 จะรับหน้าที่เป็นเซนเตอร์ค่ะ\
\
\
\
เหนือสิ่งอื่นสิ่งใด ก่อนอื่นเลย\
\
ต้องขอขอบคุณจริง ๆ นะคะ\
\
มันเต็มไปด้วยความรู้สึกขอบคุณทุก ๆ คนที่อยู่รอบตัวฉันเป็นอย่างมากเลยค่ะ\
เหล่าผู้คนที่กรุณามอบการสนับสนุนให้ ก็คือ เธอที่ตอนนี้กำลังกรุณาอ่านบล็อกเช่นนี้อยู่ เมเนเจอร์และสตาฟทุก ๆ ท่านที่สามารถพึ่งพาได้เป็นอย่างดี และเหล่าเมมเบอร์ที่นับวันสายสัมพันธ์นั้น ค่อย ๆ ลึกซึ้งมากขึ้นเรื่อย ๆ\
\
ความรู้สึกอันขอบคุณต่อผู้คนคนสำคัญของฉันที่เหลือล้นจนไม่ว่าจะสื่อออกมาเท่าไรก็ไม่มีวันหมดสิ้น ต่อจากนี้ไปจะต้องตอบแทนให้อย่างเต็มที่\
จะอุทิศทั้งแรงกายและแรงใจ พยายามอย่างสุดกำลัง จนกว่าชีวิตจะหาไม่เลยค่ะ[1] (ใน interview โนกิจูก็พูดไว้สินะคะ อันนี้)\
\
\
\
\
\
เพราะเป็น「ตอนนี้」นี่ล่ะค่ะ\
เลยคิดว่ามีบางสิ่งบางอย่างถูกฝากฝังไว้ให้กับฉัน\
ไม่คิดแบบนั้นก็คงไม่ได้ค่ะ\
\
\
เข้าใจเป็นอย่างดีเลยค่ะ ว่าต่อให้จะใช้หัวใจเลือกสรรคำพูดมามากแค่ไหน มันก็คงไม่มีทางที่จะเป็นข้อความที่สามารถคำนึงถึงความรู้สึกของทุก ๆ คนได้แน่ ๆ\
แต่ถ้าหากกรุณาอ่านแล้วล่ะก็ จะดีใจมาก ๆ เลยนะคะ !\
\
\
\
\
นับตั้งแต่วันที่ประกาศเซมบัทสึจนถึงวันนี้\
ก็ใช้ชีวิตในแต่ละวันแบบไม่สมกับเป็นตัวเองสักเท่าไรค่ะ เรื่องการเตรียมใจนั้น ก็เตรียมใจไว้ตั้งแต่ตอนที่โดนเรียกชื่อแล้ว แต่ว่า อืมม นั่นคงยังอ่อนหัดเกินไปละมั้งนะะ มันหนักหนาสาหัสมากกว่าที่ตัวเองคิดไว้เยอะเลยล่ะค่ะ\
\
ฉันน่ะ ไม่ใช่คนที่จะกล้าพูดออกมาว่า「อยากเป็นเซนเตอร์」หรอกนะคะ\
ถึงจะมีความใฝ่ฝันอยู่บ้าง แต่ทันทีที่ได้พูดออกไปจริง ๆ แล้ว\
ก็รู้สึกเหมือนว่าความไม่เอาไหนของตัวเองนั้น ถูกเปิดโปงออกมาจนหมด\
\
เพราะอย่างงั้น ที่ผ่านมาจนถึงตอนนี้\
เลยตั้งเป้าหมายไว้แค่ว่าจะ\
「ทำในสิ่งที่ตัวเองทำได้ในตอนนี้ให้เป็นอย่างดี」และ\
「ไม่ทำตัวเหลาะแหละในที่ของตัวเอง」ค่ะ\
\
\
พอได้อ่านนี่แล้ว ก็คงจะมีคนที่รู้สึกประหลาดใจอยู่บ้างสินะคะ\
ขอโทษจริง ๆ นะคะ บางทีฉันอาจจะเป็นพวกที่ชอบเอาคำพูดมาเป็นเกราะกำบังก็ได้นะคะ 55 พยายามตามหามันมาโดยตลอดเลยล่ะค่ะ วิธีการที่จะทำให้ตัวเองที่แสนอ่อนแอนั้นลุกขึ้นสู้ได้\
\
แต่ตัวฉันที่แสนอ่อนแอนั้น ก็ได้ถ้อยคำที่ตัวเองน่าจะเป็นคนร้อยเรียงมันขึ้นมาเองแท้ ๆ และภาพของทุก ๆ คนที่กรุณาตอบรับความรู้สึกจากถ้อยคำเหล่านั้น คอยมอบแรงผลักดันอย่างมหาศาลมาโดยตลอดเลยล่ะค่ะ เพื่อที่จะสามารถยืนหยัดในสถานที่อันแสนเจิดจรัสนั้นได้ ในฐานะของโนกิซากะ46\
การที่คิดจะก้าวเดินต่อไปเรื่อย ๆ บนเนินเขาอันสูงชันนั้น ด้วยตัวคนเดียวนี่ไม่มีทางเป็นไปได้หรอกค่ะ ขอบคุณทุก ๆ คนมากเลยนะคะที่พาฉันมาไกลจนถึงขนาดนี้\
\
คำพูดที่เลือกสรรขึ้นมาเพื่อมอบความกล้าให้กับใครสักคนนั้น รู้สึกตัวอีกที มันก็กลายเป็นสิ่งที่มอบพลังให้กับตัวฉันเองซะแล้วค่ะ ที่ทำให้ฉันรู้ถึงสิ่งนั้นได้ ก็เป็นเพราะว่าได้เข้ามายังโนกิซากะ และสามารถฝากฝังถ้อยคำเหล่านั้นไว้ในที่แห่งนี้ได้\
\
เพราะงั้น ฉันก็เลยชอบการเขียนบล็อกนี่ ที่ซึ่งเนื้อหาคือตัวอักษรเป็นหลักค่ะ แล้วก็ได้เลือกมันมาเป็นหัวข้อหลักของนิทรรศการเดี่ยว[2]ด้วยนะคะ\
\
\
\
\
\
\
ตัวฉันที่เข้ามายังโนกิซากะเมื่อตอนอายุ 19 และได้รับโอกาสให้สามารถเรียนต่อในระดับมหาวิทยาลัย\
\
และก้าวเดินอยู่บนเส้นทางที่เหล่ารุ่นพี่ได้พยายามอย่างสุดชีวิตเพื่อสร้างมันขึ้นมาตลอดช่วงเวลาอันแสนยาวนานของกรุ๊ป พร้อม ๆ กับคอยปัดป้องลูกศรที่พุ่งทะยานเข้ามาหาให้อยู่เสมอ\
\
\
ฉันเคยคิดถึงขั้นอยากจะใช้สถานะของตัวเองที่ได้รับโอกาสให้ใช้ชีวิตตามที่คิดตามที่หวังไว้นั้น เป็นอาวุธประจำตัวไปเลยด้วยซ้ำค่ะ และการที่ตัวฉันนั้นแตกต่างจากคนรอบข้าง บางครั้งมันก็เป็นสายลมส่งให้ตัวฉันพุ่งทะยานไปข้างหน้า แต่บางครั้งมันก็บดขยี้หัวใจของฉันอย่างไร้ความปรานีเหมือนกันค่ะ\
ทั้งวันที่ได้นั่งดูเพื่อนร่วมรุ่นของตัวเองขึ้นเวทีเป็นครั้งแรกอยู่ตรงทางหนีไฟโรงเรียนกวดวิชา[3] ทั้งความเร่าร้อนของอันเดอร์ไลฟ์ที่ได้สัมผัสเป็นครั้งแรก[4] ทั้งหน้าต่างที่ลากยาวจนไปถึงเพดานของตึกคณะจิตรกรรม ทั้งอังกอร์ที่โยโกะอาริ[5] คงไม่มีวันลืมสิ่งเหล่านี้แน่นอนค่ะ\
\
แล้วเรื่องอายุล่ะจะว่ายังไง หรือเรื่องจะที่ว่าไม่มีเวลาแล้วอะไรนั่น\
เรื่องพวกนั้นน่ะ ชีวิตคนเรามันก็เหมือนกันไม่ใช่เหรอคะ\
\
\
\
\
ฉันจะพยายามเพื่อให้ทุกคนยอมรับ และฉันจะต้องเป็นคนในแบบที่ทำให้คนอื่นรู้สึกอยากจะฝากฝังสิ่งต่างๆ ไว้ให้ได้ค่ะ

เพื่อที่จะไม่ให้ผิดหวังในภายหลัง จะพยายามนะคะ\
เพื่อที่จะให้ทุกคนยอมรับ ก็จะพยายามค่ะ จะต้องเป็นคนที่คู่ควรมากพอ ที่จะทำให้ทุกคนรู้สึกอยากฝากฝังอะไรบางอย่างไว้ให้ได้เลยค่ะ\
\
การที่จะได้รับเลือกให้มาเป็นเซนเตอร์ในช่วงเวลาแบบนี้นั้น มันก็เป็นอะไรที่ไม่ค่อยจะเกิดขึ้นได้บ่อยนัก...\
คงอาจจะพิจารณามามากไม่ใช่น้อยเลยกว่าที่จะให้ฉันมาอยู่ตรงนี้\
\
จะไม่ทำให้ผิดหวังค่ะ\
\
\
\
ทั้งความคิดแรกสุดที่แว๊บเข้ามาในหัว และทั้งสิ่งที่อยากจะเขียนมันออกไป ก็คือความรู้สึกขอบคุณถึงทุกท่านที่อยู่รอบข้าง\
นั่นก็เพราะว่า ตัวฉันได้ใช้ชีวิตอยู่ในสภาพแวดล้อมที่รายล้อมไปด้วยความรักและความเมตตาจริง ๆ ค่ะ\
\
ถ้าหากฉันได้พยายามอย่างเต็มที่ในจุด ๆ นี้ บางทีมันอาจจะกลายเป็นบางอย่างที่สามารถส่งต่อไปสู่ใครสักคนก็เป็นได้\
\
\
\
เมมเบอร์อีกหลายคนก็มีในสิ่งที่ตัวฉันไม่มี\
ฉันอยากจะเป็นเซนเตอร์ ที่สามารถดึงเอาความเปล่งประกายของสิ่ง ๆ นั้นในแต่ละคนออกมาให้ได้ค่ะ\
\
