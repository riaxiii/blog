_[รูป 1]_\
\
\
\
ช่วงไม่กี่วันมานี้ สายลมเย็น ๆ ก็เริ่มโบกพัดมา\
เป็นช่วงเวลาที่เริ่มจะรู้สึกว่า ฤดูร้อนกำลังค่อย ๆ จางหายไปค่ะ\
\
\
ทั้ง ๆ ที่เพิ่งผ่านการแสดงรอบสุดท้ายมาเมื่อไม่กี่วันแท้ ๆ\
แต่วิวทิวทัศน์ที่ได้เห็นเมื่อวันเวลาเหล่านั้น เหมือนกลับกลายเป็นภาพมายาที่ไม่สามารถจับต้องได้ ราวกับว่าได้ตกลงสู่ภาพลวงตา ที่ผ่านมานานแสนนานแล้วยังไงอย่างงั้นล่ะค่ะ\
\
เป็นช่วงเวลาที่ได้ใช้ไป อย่างกับได้ดำดิ่งสู่โลกแห่งความฝันเลยล่ะค่ะ\
\
วันนี้ ขอให้ฉันได้ส่งมอบความรู้สึกแม้เพียงเล็กน้อย ให้กับทุกท่านที่ได้ใช้เวลาร่วมกันมาในหน้าร้อนนี้เถอะนะคะ\
\
\
\
\
\
「ทัวร์ทั่วประเทศกลางฤดูร้อน 2025」\
\
\
ถึงทุกท่านที่ได้กรุณามายังสถานที่จัดงาน\
ทุกท่านที่คอยเชียร์ผ่านทางถ่ายทอดสด\
แล้วก็ทุก ๆ ท่านที่คอยส่งมอบคำพูดอันแสนอบอุ่นนั่น\
เป็นเพราะทุก ๆ ท่าน ถึงได้สามารถผ่านทัวร์ทั่วประเทศครบทั้ง 16 รอบมาได้อย่างปลอดภัยค่ะ\
ขอขอบคุณจากใจจริงเลยนะคะ\
\
\
และในครั้งนี้ ก็เป็นไลฟ์จิงกุครั้งที่ 10 ที่ควรแก่การเฉลิมฉลองล่ะค่ะ\
\
ต่อเนื่องจากปีที่แล้ว การที่ตัวเองได้มายืนในสถานที่แห่งนี้นั้น\
มันไม่ใช่เรื่องง่าย ๆ เลยค่ะ\
\
ด้วยสตาฟทุก ๆ ท่านที่คอยสนับสนุนอยู่เรื่อยมา\
แฟน ๆ ที่ทุกท่าน ที่กรุณาให้ความสำคัญและคอยส่งมอบเสียงเชียร์อันอบอุ่นแก่พวกเรา และทุก ๆ การสนับสนุนจากท่านที่เกี่ยวข้อง จึงทำให้ไลฟ์นี้สามารถเกิดขึ้นมาได้ค่ะ\
ต้องขอขอบคุณจากใจจริง สำหรับทุก ๆ สิ่งทุก ๆ อย่างเลยค่ะ\
เป็นเพราะทุก ๆ คน ปีนี้ก็เลยเป็นอีกหนึ่งความทรงจำที่จะไม่มีวันลืมเลือนค่ะ\
\
\
\
ในฤดูร้อนครั้งนี้ ฉันได้รับโอกาสให้ยืนในตำแหน่ง front ล่ะค่ะ\
ในทุก ๆ การแสดง ก็อยากจะเก็บรักษาทุกช่วงเวลาเหล่านั้น ช่วงเวลาที่ไม่สามารถมีอะไรมาแทนที่ได้\
เลยใช้ชีวิตในทุกวันด้วยความรู้สึกแบบนั้น และเผชิญหน้ากับแต่ละสิ่งแต่ละอย่าง โดยตั้งใจไว้ว่าจะทุ่มเททุกสิ่งทุกอย่างให้มากกว่าที่เคยเป็นมาค่ะ\
\
ขณะเดียวกัน ก็เป็นทัวร์ทั่วประเทศครั้งแรกในฐานะรุ่นพี่ด้วยล่ะค่ะ\
ทำอย่างไรจึงจะสามารถเติมเต็มบทบาทหน้าที่ของตัวเอง\
ให้สมกับเป็นรุ่นพี่ที่จะนำพาทุก ๆ คนได้\
ในระหว่างที่ตามหาคำตอบและได้ใช้ชีวิตอยู่ในช่วงเวลาเหล่านั้น สิ่งที่ได้พบก็คือ\
「การนำพาทุกคนผ่านการแสดงมากกว่าผ่านทางถ้อยคำ」\
และ「การเป็นตัวตนที่สามารถส่งมอบความอุ่นใจได้」ค่ะ\
\
\
ในขณะที่กำลังสัมผัสถึงสายลมลูกใหม่จากการที่มีสมาชิกรุ่น 6 เข้ามาเพิ่ม ทัวร์ครั้งนี้ก็ได้มาถึง\
การแอบซ่อนความรู้สึกกังวลและความขัดแย้งเอาไว้ในใจ และนำพาทุก ๆ คนผ่านการแสดงและเป็นตัวตนที่สามารถส่งมอบความอุ่นใจได้\
ฉันคิดว่านั่นคือ บทบาทหน้าที่ที่ดีที่สุดที่ฉันสามารถทำมันได้ค่ะ\
\
\
\
พอคิดย้อนกลับไปแล้ว ฤดูร้อนครั้งนี้เป็นอะไรที่ต้องลองผิดลองถูกในทุก ๆ เลยล่ะค่ะ\
หลังจากจบไลฟ์แล้วก็จะมานั่งทบทวนและหาทางแก้ไขในสิ่งที่ตัวเองทำพลาดภายในวันนั้นเลย\
พอเช้าวันถัดมาก็จะพบกับปัญหาใหม่ ๆ\
และตามหาคำตอบนั่นอยู่เสมอมา ว่าทำอย่างไรถึงจะสามารถเข้าใกล้การแสดงออกในอุดมคติให้ได้มากกว่านี้\
\
\
\
\
แม้ในวันที่ไม่มีไลฟ์ ก็ฟังเพลงโนกิซากะซ้ำไปซ้ำมา\
แล้วก็ลองแสดงมันออกมาดู โดยจินตนาการว่าหากตัวเองเป็นคนที่อยู่ในโลกของบทเพลงนั่นแล้ว จะรู้สึกอย่างไรกันแน่นะ\
ตั้งเป้าหมายไว้ว่าจะ「ไม่ว่าจะไลฟ์ที่ไหนก็จะทำให้เป็นไลฟ์ที่สนุกสนานด้วยความรู้สึกอันสดใหม่」วันเวลาที่ค่อย ๆ มุ่งหน้าเข้าสู่การแสดงรอบสุดท้ายที่จิงกุทีละเล็กทีละน้อย\
สำหรับฉันแล้ว นั่นคือประสบการณ์ที่ไม่มีอะไรสามารถมาทดแทนได้ค่ะ\
\
\
\
\
\
\
\
\
\
ถึงมิคุ\
\
ฉันชอบความสัมพันธ์ที่ถึงแม้จะไม่ได้พูดอะไรออกมา แค่มองตากันก็ต่างฝ่ายต่างรู้สึกได้ถึง condition ของวันนั้นได้ แล้วก็คอยเติมเต็มในสิ่งที่ต้องการซึ่งกันและกันค่ะ\
ทั้งการที่สามารถสื่อสารความรู้สึกของตัวเองออกมาเป็นคำพูดได้อย่างชัดเจน\
ทั้งการที่สามารถมองเห็นถึงข้อดีมากมายของคนที่อยู่รอบข้าง\
ทั้งการที่สามารถยอมรับในสิ่งนั้นได้อย่างตรงไปตรงมา\
\
สำหรับฉันแล้ว มิคุคือตัวตนที่ราวกับท้องฟ้าเลยล่ะค่ะ ถึงจะดูเหมือนเอื้อมมือไปถึงได้ แต่ก็ไปไม่ถึง\
แต่ว่านะ ก็รู้สึกประหลาดใจเหมือนกัน พอได้ยืนมองมิคุที่อยู่ข้าง ๆ บนสเตจแล้ว ก็รู้สึกว่าตัวเองยังสามารถพยายามต่อไปได้ไม่ว่าจะอีกมากแค่ไหน\
และไม่ว่าจะอีกไกลแค่ไหนก็อยากที่จะไล่ตามความฝันด้วยกันค่ะ\
การที่ได้พบเจอกับตัวตนเช่นนี้แล้วนั้น มันเป็นอะไรที่น่ามหัศจรรย์จริง ๆ เลยค่ะ\
นหน้าร้อนครั้งนี้ที่ได้ยืนคู่กับมิคุนั้น เป็นเรื่องที่ยอดเยี่ยมสุด ๆ ไปเลยล่ะค่ะ\
พวกเราทำได้แล้วนะ ! ! !\
ขอบคุณมาก ๆ เลยนะ\
\
\
ถึงคาคิซัง\
\
นางเอกแห่งฤดูร้อนครั้งนี้ คาคิซัง\
ภาพลักษณ์ที่ทั้งเปล่งประกาย ตรงไปตรงมา และคอยนำพาพวกเราอยู่เสมอ โดยที่ไม่เคยแสดงให้เห็นถึงความกังวลใจให้พวกเราที่เป็นรุ่นน้องได้เห็นเลยนั้น ทำให้รู้สึกอุ่นใจเป็นอย่างมากเลยล่ะค่ะ\
\
ในขณะที่เป็นคนที่ยืนหยัดอยู่ข้างหน้าและคอยมอบความอุ่นใจอยู่เสมอ บางครั้งก็เหลือบไปเห็นสีหน้าที่ไม่ค่อยสบายใจนักอยู่บ้าง ก็เลยคิดว่าอยากจะเป็นคนที่สามารถรับรู้ได้ถึงท้องฟ้าอันมืดครึ้มของคาคิซังบ้าง\
และก็รู้สึกว่าอยากจะแข็งแกร่งขึ้น เป็นคนที่สามารถแบ่งเบาภาระที่กำลังแบกรับไว้อยู่ได้บ้าง แม้จะเพียงเล็กน้อยก็ตาม\
\
คาคิซังคือดวงอาทิตย์ของฉันค่ะ\
พอได้มองดูรอยยิ้มของคาคิซังแล้ว ก็รู้สึกว่าคำพูดที่ว่า「รอยยิ้มคือสิ่งที่จะเยียวยาโลกใบนี้」นั้น เป็นเรื่องจริงล่ะค่ะ\
ฤดูร้อนที่มีคาคิซังที่เป็นดั่งดวงอาทิตย์ที่คอยอยู่เคียงข้างมาโดยตลอดนั้น\
สำหรับฉันแล้ว ทุก ๆ วันก็เป็นเหมือนวันท่ามกลางฤดูร้อนล่ะค่ะ\
\
\
\
การที่ได้รับโอกาสให้ยืนเคียงข้างคาคิซังนั้น\
สำหรับฉันแล้วเป็นเกียรติอย่างมากเลยค่ะ\
ขอบคุณมาก ๆ เลยนะคะ ! ! !\
\
\
\
\
\
\
ถึงสตาฟทุก ๆ ท่าน\
\
สตาฟทุก ๆ ท่านที่คอยจัดเตรียมอะไรหลาย ๆ อย่างให้ตั้งแต่เช้าตรู่ก่อนที่พวกเราจะมาถึง และคอยทบทวนสิ่งที่ทำไปให้ซ้ำไปซ้ำมาจนถึงดึกดื่น\
จัดการเรื่องกล้อง แสงสี จนไปถึงการออกแบบเสื้อผ้า\
คอยใฝ่หารูปแบบวิธีการที่จะทำให้พวกเราเปล่งประกายให้ได้มากที่สุด สัมผัสได้ถึงความรักอยู่ทุกหนแห่งตลอดเวลาเลยล่ะค่ะ\
\
ขอขอบคุณเป็นอย่างมากเลยนะคะ\
\
\
\
ท่ามกลางอากาศร้อนระอุ เวลาที่ได้เห็นช่างกล้องในช่วงซ้อม\
ถือผ้าเชียร์ของเมมเบอร์ไว้คนละ 1 ผืน และคอยเชียร์อยู่นั้น\
ก็เป็นอีกหนึ่งครั้งที่รับรู้ได้ว่า การดำเนินกิจกรรมต่าง ๆ ของโนกิซากะ46นั้น ยังคงอยู่ได้เพราะการสนับสนุนอย่างมากมายจากหลาย ๆ ท่านค่ะ\
\
\
\
\
\
และเหนือสิ่งอื่นใด แฟน ๆ ทุกท่าน\
\
\
การที่ตอนนี้พวกเรามายืนอยู่ในสถานที่แห่งนี้ในฐานะไอดอลได้นั้น ไม่ต้องสงสัยเลย เป็นเพราะแฟน ๆ ทุกท่านที่คอยกรุณามอบความรักและเสียงเชียร์อยู่นั่นล่ะค่ะ\
ในระหว่างที่แสดงอยู่ท่ามกลางสภาพอากาศที่ร้อนจัด ภาพที่ทุก ๆ ท่านคอยส่่งมอบความรักอย่างสุดกำลังโดยที่ไม่หยุดหย่อนนั้น มันช่างอิ่มอกอิ่มใจเป็นอย่างมากเลยค่ะ\
ทุก ๆ คนที่กำลังกรุณาอ่านบล็อกนี้อยู่ ต่างก็มีเส้นทางชีวิตแตกต่างกันไป และทุกคนนั้น ก็คือตัวเอกของพวกเราค่ะ !\
เหมือนกับที่แฟน ๆ ทุกท่านทำให้พวกเราได้เป็นตัวเอกในฤดูร้อนนี้\
ต่อจากนี้ก็ ขอให้พวกเราได้ทำให้ทุก ๆ คนมีความสุขด้วยเถอะนะคะ☺️\
\
\
มันคือเป้าหมายที่ตั้งไว้ของฤดูร้อนนี้ล่ะค่ะ\
สัญญาไว้ว่าจะทำให้ทุก ๆ คนมีความสุข\
ดีใจมาก ๆ เลยล่ะ ที่สามารถรักษาสัญญาเอาไว้ได้\
หวังว่าจะรักษามันไว้ได้ต่อไปเรื่อย ๆ นะ\
ฉันจะดูแลเอาใจใส่คุณตลอดไปค่ะ\
\
\
\
\
\
\
\
\
และตั้งแต่วันนี้ จะเริ่มฉายถ่ายทอดสดย้อนหลังจิงกุ 4 วันค่ะ !\
ถ้าหากได้กรุณารับชมสเตจบทสรุปของฤดูร้อนครั้งนี้ซึ่งเต็มเปี่ยมไปด้วยความรักอันเหลือล้นนี้ละก็ จะรู้สึกดีใจมาก ๆ เลยค่ะ\
\
แล้วก็อีกครั้งหนึ่ง ขอขอบคุณทุกท่านที่คอยอยู่เคียงข้างด้วยในฤดูร้อนครั้งนี้ จากหัวใจเลยนะคะ !\
ขอบคุณมาก ๆ เลยนะคะ ! !\
