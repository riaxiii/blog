\
สวัสดีค่ะ !\
คาวาซากิ ซากุระ ค่ะ\
\
\
\
\
เวลาที่ออกไปข้างนอกแล้ว\
อากาศที่เริ่มจะเย็นขึ้นก็สัมผัสกับแก้ม\
นั่นก็ทำให้รู้สึกว่า ฤดูกาลกำลังเข้าก้าวออกไปสู่ฤดูหนาวอย่างเงียบ ๆ แล้ว\
\
\
เสื้อผ้าฤดูใบไม้ร่วงก็ยังไม่ทันจะได้เอาออกมาใส่\
เผลอแปปเดียวปีนี้ก็ดูเหมือนฤดูหนาวจะมาถึงซะแล้ว\
\
\
\
ลมหนาวที่พัดมาจนต้องรู้สึกเสียวสันหลังวาบอยู่ตลอดเวลา\
ไฟประดับที่ส่องสว่างอยู่ตามถนนหนทาง\
แสงจากเสาไฟต้นหนึ่งที่กำลังทำให้รอยยิ้มของใครสักคนเปล่งประกายยิ่งขึ้น\
ชอบหน้าหนาวมากเลยล่ะ\
\
\
\
\
ถ้าหากปีนี้ได้อยู่กับเธอซึ่งเป็นคนพิเศษในฤดูหนาวนี่ด้วยกันล่ะก็ คงจะดีไม่น้อยเลยนะ..☺︎\
\
\
\
\
\
\
\
ในซิงเกิ้ลลำดับที่ 40 ก็รับเลือกให้ดำเนินกิจกรรมในฐานะเซมบัทสึเมมเบอร์ค่ะ\
\
และนั่นก็เป็นเพราะทุก ๆ คนที่คอยสนับสนุนอยู่เสมอมาค่ะ\
ขอขอบคุณมาก ๆ เลยนะคะ\
\
ในครั้งนี้ได้อยู่ใน position「fukujin[1]」ค่ะ\
จะจดจำความสำคัญนี้ไว้หัวใจ\
และก็อยากจะดำเนินกิจกรรมไปด้วย พร้อม ๆ กับยืนยันกับทุกความหมายแต่ละความหมายของการที่ตัวเองได้มายืนอยู่ในที่แห่งนี้\
\
\
\
\
\
เซนเตอร์ของซิงเกิ้ลที่ 40 นั้น\
เป็นรุ่นที่ 6 ยาดะ โมเอกะ จัง กับ เซโตกุจิ มิทสึกิ จังค่ะ\
\
พอได้ยินว่าเป็นดับเบิ้ลเซนเตอร์แล้วก็\
หน้าของรุ่นพี่ซึ่งที่รักอย่างยิ่งก็ลอยขึ้นมาเลยล่ะค่ะ\
\
\
ในระหว่างที่กำลังคุยกับรุ่นพี่ที่เคารพรักอยู่นั้น\
ฉันเองก็คิดอยู่ในใจไม่รู้กี่ครั้งต่อกี่ครั้งเลยล่ะค่ะ\
ว่าถ้าหากสามารถอยู่เคียงข้างคอยสนับสนุนได้ละก็\
\
\
สิ่งที่เคยได้รับมา ก็อยากจะส่งมอบให้ต่อ ๆ ไป\
และแบ่งปันความรู้สึกกับทั้ง 2 คนค่ะ\
\
\
\
เมื่อใดที่ทั้ง 2 คน\
ถูกกดดันด้วยภาระอันหนักอึ้งที่ต้องแบกรับไว้\
ก็อยากจะเป็นคนที่คอยยื่นมือเข้าไปช่วยเหลืออย่างอ่อนโยน\
ไม่ว่าเมื่อไร\
ก็อยากจะเป็นมิตรแท้ที่คอยเคียงข้างอยู่เสมอค่ะ\
\
\
และ เพื่อที่จะให้คุโบะซังที่คอยสนับสนุนโนกิซากะ46จนมาถึงตอนนี้ สามารถมุ่งสู่วันจบการศึกษาได้อย่างสบายใจ\
ก็อยากจะให้เป็นช่วงเวลาที่ได้เติบโตไปอีกหนึ่งขั้นพร้อม ๆ กับทุกคนค่ะ\
\
\
\
ในช่วงซิงเกิ้ลที่ 40 นี้\
หวังว่ามันจะเป็นฤดูกาลที่มีความหมายสำหรับโนกิซากะ46นะคะ\
\
\
\
\
\
\
\
\
\
เครื่องแบบที่เต็มเปี่ยมไปด้วยความทรงจำ\
\
\
_[รูป 1 ชุดเครื่องแบบประจำซิงเกิ้ลของ same numbers]_\
\
\
\
เพราะว่าได้รับความช่วยเหลือจากการมีอยู่ของเธออยู่ฝ่ายเดียวมาโดยตลอด\
ไม่ใช่แค่ความฝันของฉันเพียงคนเดียว\
ฉันเองก็ อยากเป็นคนที่ช่วยทำให้ความฝันของเธอเป็นจริงไปพร้อม ๆ กันด้วยนะ\
จะดีใจมาก ๆ เลยล่ะ ถ้าหากมาพึ่งพาฉันบ้างในเวลาที่เธอกำลังเป็นกังวลอยู่ตัวเพียงคนเดียว\
จะเรื่องเล็กเรื่องน้อยขนาดไหนก็อยากจะคุยด้วยเยอะ ๆ นะ\
\

\
