ไม่ได้เจอกันนานเลยนะคะ\
\
\
คาวาซากิ ซากุระ ค่ะ\
\
\
_[รูป 1]_\
\
\
วันนี้อยากจะมาเขียนเรื่องที่เกิดขึ้นในช่วงนี้น่ะค่ะ☺︎\
\
\
ก่อนอื่นเลยก็ ยังไงก็ต้องเป็นเรื่องนี้เลย !\
\
\
โนกิซากะ46ได้รับเลือกให้ขึ้นแสดงในงาน\
NHK Kouhaku uta gassen ครั้งที่ 76 ค่ะ !\
\
\
\
ต้องขอขอบคุณเป็นอย่างมากเลยนะคะ\
\
\
\
\
\
ตอนที่รู้ว่าจะได้ขึ้นแสดงในงาน kouhaku\
อุเมะซังที่แทบจะไม่เคยร้องไห้ให้ใครเห็นเลย ก็ร้องไห้ออกมา\
ราวกับว่าสิ่งที่แบกมารับมาโดยตลอดนั้น ได้ถูกวางลงอย่างอ่อนโยน\
「พวกเรา พยายามกันมาอย่างหนักเลยนี่เนอะ」คำพูดของอุเมะซังที่มอบให้กับรุ่นน้องในตอนที่พวกเรากำลังล้อมรอบอุเมะซังกันอยู่นั้น\
มันได้แทรกซึมลึกลงไปในหัวใจ และจะไม่มีวันลืมเลยค่ะ\
\
\
_[รูป 2]_\
\
\
รุ่นพี่รุ่นที่ 4 ที่ได้เห็นฉากนั้น\
ก็ดีใจจนร้องไห้ออกมา และนั่นก็ทำให้ฉันรู้สึกประทับใจจริง ๆ ค่ะ\
พอคิดว่าเหล่ารุ่นพี่ที่พึ่งพาได้มาโดยตลอดนั้น จริง ๆ แล้วก็รู้สึกกังวลมาตลอดทั้งปีเหมือนกัน ก็ทำให้รู้สึกอิ่มเอมหัวใจมาก ๆ เลยค่ะ\
\
\
\
_[รูป 3]_\
\
\
พอรวบรวมความกล้าแล้วเดินเข้าไปคุยกับอุเมะซัง\
ก็ได้ถ่ายรูปทูช็อตด้วยกันเป็นครั้งแรกด้วย\
ความทรงจำล่ะค่ะ\
\
\
\
\
ตอนที่ได้ขึ้นแสดง kouhaku ครั้งแรกเมื่อปี 2022\
ตัวเองที่ยังไม่ทันจะรู้เรื่องอะไรเลยในตอนนั้น ในปีนี้ก็จะมุ่งหน้าสู่เวทีอันสุดพิเศษแห่งนี้ ด้วยความรู้สึกที่แตกต่างออกไปจากตอนนั้นค่ะ\
และอยากจะขึ้นไปยืนบนจุดนั้น โอบกอดความหมายของการก้าวเดินจนมาถึงตอนนี้ และไม่ลืมความรู้สึกขอบคุณค่ะ\
\
\
\
\
\
\
\
\
คิดว่าคงจะยุ่ง ๆ จนไปถึงสิ้นปีเลยค่ะ\
แต่ถ้าหากได้วิ่งฝ่าฟันจนไปถึงท้ายที่สุดกับทุก ๆ คนที่ฉันรักแล้วล่ะก็ คงน่าดีใจไม่น้อยเลยค่ะ\
\
\
\
\
ขอบคุณมาก ๆ เลยนะคะ\
ที่คอยรักและสนับสนุนพวกเรามาตลอดโดยไม่เปลี่ยนแปลง\
เพราะทุก ๆ คนนั่นล่ะค่ะ ถึงได้มายืนอยู่ในเวทีแห่งนี้อีกครั้ง\
\
\
