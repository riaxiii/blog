\
_[รูป 1]_\
\
\
\
ปี 2025 ก็ต้องขอขอบคุณที่คอยอยู่เคียงข้างด้วยนะคะ☺︎\
ปีนี้เป็นหนึ่งปีที่ความฝันหลาย ๆ อย่างเป็นจริงล่ะค่ะ\
\
\
\
\
วันนี้ก็เลย\
จะลองมองย้อนกลับไปดูเรื่องที่เกิดขึ้นในปีนี้กันสักหน่อย\
ท่านไหนที่พอจะมีเวลา ถ้าได้อ่านแล้วก็จะดีใจเป็นอย่างมากเลยค่ะ🪽\
\
\
\
ก่อนอื่นเลย รายการ「dai 67 kai kagayaku ! nihon record taishou[1]」เมื่อวาน\
เป็นครั้งแรกเลยที่ได้มายืนบนเวทีนี้\
ตั้งแต่ตอนซ้อมก็แบบว่า น้ำตาคลอซะแล้วล่ะค่ะ\
\
\
\
ตอนที่แบบ ตรงนี้ถ้าทำแบบนี้จะดีกว่านะ\
เวลาที่ลงรายละเอียดในจุดเล็ก ๆ ด้วยกันกับทุกคนนั้น มันสนุกมากเลยล่ะ\
แล้วก็รู้สึกว่าตอนที่มันเริ่มจะสมบูรณ์มากขึ้นทีละเล็กทีละน้อยนั้น\
ก็น่าดีใจมาก ๆ เลยล่ะะ\
\
\
หลังจากที่พวกเราล้อมวงบูมกันเสร็จแล้ว\
Seishiro ซังก็บอกว่า\
「สไตล์นี้ พวกเธอคือออริจินัลนะ เพราะงั้นก็จงภูมิใจในตัวพวกเธอเอง แล้วก็ออกไปลุยกันเถอะ」\
นั่นเป็นคำพูดที่ ทำให้รู้สึกดีใจสุด ๆ เลยล่ะค่ะ\
\
\
\
ความรู้สึกอันหนักหน่วงและตึงเครียดที่ได้สัมผัสในตอนนั้น\
ต่อจากนี้ก็อยากจะดูแลมันไว้ให้เป็นอย่างดีค่ะ\
\
\
\
ส่วนตัวแล้ว\
เป็นหนึ่งปีที่ได้สัมผัสกับประสบการณ์อันแสนล้ำค่าค่ะ !\
\
\
หน้าร้อนปีนี้ก็ได้ยืนตำแหน่ง front เป็นครั้งแรก\
ในทุก ๆ วันก็ได้พยายามอย่างเอาเป็นเอาตาย\
และก็ใช้ชีวิตในแต่ละวันเวลาด้วยการเตรียมใจที่ว่า\
「จะทำให้เป็น 1 วันที่ต่อให้วันนี้จะเป็นวันสุดท้ายก็จะไม่เสียดาย」\
\
\
คำพูดที่ทุกคนบอกว่า\
เป็นหน้าร้อนนี้ที่มีความสุขมาก ๆ เลยล่ะ\
มันแบบ จึ้ก ลงไปในหัวใจ แล้วก็รู้สึกอบอุ่นขึ้นมาเลยล่ะ\
\
\
อีกเรื่องที่รู้สึกขอบคุณมาก ๆ ก็คือได้แสดง CM เดี่ยวด้วยล่ะ\
แล้วก็การที่ได้เดินบนรันเวย์ที่ใฝ่ฝันมาด้วยตลอดก็ด้วย !\
\
\
\
ถึงจะรู้สึกเครียดมากขนาดไหนเวลาที่อยู่บนรันเวย์\
แต่พอได้ยินเสียงจากทุก ๆ คนแล้ว\
ก็รู้สึกว่ามันสนุกมาก ๆ เลยล่ะ !\
\
\
เพราะว่าทุกคนคอยอยู่เคียงข้าง\
ก็เลยก้าวข้ามผ่านปีนี้ไปได้ค่ะ\
ต้องขอขอบคุณมาก ๆ เลยนะคะ☺️\
\
\
ปีหน้าก็\
อยากจะเป็นคนที่สามารถเป็นพลังให้กับโนกิซากะ46ให้ได้มากกว่านี้ค่ะ\
\
\
อยากจะลองทำงานที่เกี่ยวกับละครดราม่าหรือภาพยนตร์ดูบ้าง\
งานที่เกี่ยวกับนางแบบก็อยาก\
ถ้าสักวันหนึ่งได้แสดง MV ของศิลปินที่ชอบมาก ๆ ล่ะก็คงดีเนอะ\
\
\
\
\
\
ยังมีความฝันอีกมากมายที่ยังไม่ถูกเติมเต็มอีกเพียบเลยค่ะ !\
\
มีแต่เรื่องที่อยากทำเต็มไปหมดเลยล่ะค่ะ\
\
\
\
หลังจากนี้「kouhaku uta gassen ครั้งที่ 76」ก็\
อยากจะแสดง performance ออกมาด้วยความเอาใจใส่\
และด้วยความรู้สึกขอบคุณในทุกคนที่คอยสนับสนุนมาโดยตลอดค่ะ\
\
\
ถ้าเช่นนั้นก็\
ขอให้เป็นปีที่ดีนะคะ\
\
\
\
ตอนแรกบอกว่านิดเดียว\
แต่ก็ดันเขียนยาวซะขนาดนี้ไปซะได้..\
ขอโทษด้วยนะ 555\
\
\
\
\
\
ช่วง 1 ปีนี้ก็ ขอขอบคุณที่เหน็ดเหนื่อยกันมานะ☺︎\
พยายามอย่างมากในทุก ๆ วันเลยสินะ\
เก่งมากเก่งมาก☺️\
ปีหน้าก็\
ถ้าอยู่ด้วยกันข้าง ๆ แล้วล่ะก็ จะดีใจมาก ๆ เลยล่ะ\
\
\
\
\
รักมาก ๆ เลยนะ !\
\
\
\
\
\
อ่อ แล้วก็นะ\
ซัทสึกิกับมิคุที่อยู่ข้าง ๆ ก็\
ในระหว่างที่บอกว่าเป้าหมายปีหน้าคือ「ดูเป็นผู้ใหญ่」ก็กำลังนั่งเลือกเครื่องประดับลุคพี่สาวกันอยู่น่ะค่ะ\
เป็นรองกัปตันและเพื่อนร่วมรุ่นที่น่ารักจริง ๆ เลยค่ะ🍵\
\
\
\
ไว้จะมาเขียนใหม่นะ\
\
\
\
ของแถม\
